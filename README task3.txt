# Books REST API

 Project Description
This Node.js and Express REST API manages a list of books stored in memory. It supports Create, Read, Update, and Delete (CRUD) operations on books through HTTP endpoints.

 How to Run
1. Run npm install to install dependencies.
2. Run npm start to start the server on port 3000.
3. Use an API client like Postman to test the endpoints.

 API Endpoints
- GET /books - Get all books
- POST /books - Add a book with JSON body { "title": "...", "author": "..." }
- PUT /books/:id - Update book by ID
- DELETE /books/:id - Delete book by ID
