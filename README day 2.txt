To-Do List Web App

Project Description
This is a simple To-Do List web application created as part of the Elevate Labs web development internship task. Users can add new tasks, mark tasks as complete or incomplete, and remove tasks dynamically without page reload.

 Tools Used
- HTML for page structure
- CSS for styling
- Vanilla JavaScript for dynamic behavior (DOM manipulation, event handling)

 Features
- Add new tasks dynamically
- Mark tasks complete/incomplete by clicking on tasks
- Remove tasks using the Remove button
- Instant UI updates without page reload

 How to Run
1. Clone or download the repository
2. Open index.html in a web browser
3. Use the input box and Add button to create tasks
4. Click tasks to toggle completion
5. Click Remove button to delete tasks
