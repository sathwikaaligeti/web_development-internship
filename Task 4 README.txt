Mobile-Friendly Website Using Media Queries

Project Description
This project involves making an existing desktop website mobile-friendly by adding CSS media queries. The goal is a responsive design that looks great on both desktop and mobile devices.

Techniques Used
- CSS media queries targeting max-width 768px
- Flexible container widths
- Stacking layout elements vertically on small screens
- Responsive images and font sizes
- Navigation menu adjustments for mobile

How to Test
Open the website in a browser and use Chrome DevTools device toolbar to simulate various mobile screen sizes.
 Files Included
- index.html
- style.css (with media queries)
